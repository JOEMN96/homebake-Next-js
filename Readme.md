## Stack Used

- Frontend - Next.js, Redux, React-query, Ant-d, Scss 
- CMS - Strapi, mongoDB
- Backend - Node.js, Express, MongoDB, Stripe

## Details

- Password are Encrypted and salted.
- Cloudinary is used to host images for CMS.
- Front end and CMS are hosted in heroku

#### backend

- PW 

#### CMS

- CLOUDINARY_KEY
- CLOUDINARY_NAME

- DATABASE_NAME
- DATABASE_URI
